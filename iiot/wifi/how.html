  • Hotel

Step 4. Return Easily

  • 1 - Pack the following items into the pouch

    Wi-Fi Router

    AC Adaptor

    USB Cable
    *Please make sure to put all the items into the pouch.

  • 2 - Place it into the provided envelope

    Seal up the envelope(Letter Pack) to ensure it does not fall out.

  • 3 - Drop the envelope into post box

    Simply drop the provided envelope into any Japan Post box.

    *If you would like to return at the airport, please return it BEFORE passing the security check.

    *Please note that there is no post box after you go through
    the security check and it CANNOT be returned by other company's return box.
